# Cinema Site
Website of a Cinema created with HTML, CSS, JAVASCRIPT and Node.js

## Site content
* Home page
* Pages corresponding to each movie running at the cinema
* Cinema program
* Cinema menu
* Page 404
* Sign In, Sign Out page
* Photo gallery

#### The site is responsive - phone, tablet or desktop
